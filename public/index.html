Russia army casualties in Russian-Ukrainian War (Parser)

A cozy opensource Russia army casualties parser for open-source lovers. Parses, stores and publishes structured russian army losses. Updated daily at 8:30 UTC (10:30 Europe/KIEV)

To access and modify resources, you can use any HTTP method:

GET POST PUT PATCH DELETE OPTIONS

API Request

Today's casualties 

curl -fsSL ...

Response structure example:

"day": 516,
casualties: {
  "militaryPersonnel": 242620,
  "jet": 315,
  "copter": 310,
  "tank": 4162,
  "armoredCombatVehicle": 8118,
  "artillerySystem": 4675,
  "airDefenceSystem": 452,
  "mlrs": 697,
  "supplyVehicle": 7182,
  "ship": 18,
  "uav": 3963

Parser built with: NodeJS, Pupetter.js, Supabase, Github Actions
